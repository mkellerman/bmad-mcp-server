## Implementation Notes

### Data Structure Mapping

```typescript
ChatConversation.messages[] ‚Üí Steps[]

Step mapping:
- message.role = 'system' ‚Üí Step: "‚öôÔ∏è System Prompt"
- message.role = 'user' ‚Üí Step: "üë§ User Input #N"
- message.role = 'assistant' (with toolCalls) ‚Üí Step: "ü§ñ Assistant Response #N (tool call)"
- message.role = 'tool' ‚Üí Step: "üîß Tool Execution"
- message.role = 'assistant' (text only) ‚Üí Step: "ü§ñ Assistant Response #N"
```

### Step Numbering

- Number user inputs: "User Input #1", "User Input #2"
- Number assistant responses per user turn: "Assistant Response #1", "Assistant Response #2"
- Keep tool executions unnumbered (they're sub-steps)

### Collapsed/Expanded State

- Default: All steps VISIBLE but content TRUNCATED
- Click step to expand full content
- Persist expansion state in sessionStorage
